# Hospital Room Management System

## Overview

A cloud-based hospital room management system developed using Node.js, Express.js, MySQL, and EJS.

The system allows hospitals to:

- Manage patients
- Manage rooms
- Assign patients to rooms
- Monitor isolation priorities
- Control role-based access
- Generate notifications and activity feeds

---

## Features

- Authentication system
- Role-based access control
- Patient management
- Room management
- Assignment system
- Notification system
- Responsive dashboard
- Theme persistence
- Search/filter/pagination
- Cloud deployment
- CI/CD integration
